Sobre el código:

Esta aplicación basada en Java utiliza el framework Spring para integrar y consumir diversas API literarias, facilitando la búsqueda de libros por título, la visualización de listas de libros y autores registrados, así como la capacidad de filtrar libros por idioma. Este enfoque no solo mejora la accesibilidad a la información literaria, sino que también ofrece una solución integral para la gestión y el análisis de datos relacionados con libros.

Mi experiencia incluye el desarrollo en el entorno de desarrollo Intellij IDEA, donde he optimizado el código y asegurado la escalabilidad y robustez del sistema. A través de estas tecnologías y metodologías, he contribuido a la creación de una herramienta poderosa para entusiastas de la literatura.

Tecnologías: Java, Intellij IDEA
